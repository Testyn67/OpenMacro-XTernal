## What is XTernal?

XTernal is a free, open source external macro for the Roblox game **Fisch**. Unlike traditional pixel search based macros, XTernal reads Roblox's internal memory directly making it more reliable, more consistent, and harder to desync from the game's actual state.

## What is OpenMacro?

OpenMacro is a software brand building automation tools for Roblox. XTernal is our free, open source offering for the community and is actively maintained.

## How does it work?

XTernal operates externally to Roblox, reading game memory to react to real ingame state rather than guessing based on timers or screen capture. This gives it a significant consistency advantage over traditional macro approaches.
